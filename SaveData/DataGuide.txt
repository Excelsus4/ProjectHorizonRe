====데이터====
Index: 아이템 코드, 순서대로 작성 권장, 변경할경우 인벤토리에 문제생길수 있음
Name: 게임에 표시되는 아이템 이름 문자열
Sprite IDX: 게임에 표시되는 그래픽 인덱스
Part IDC: 해당 파츠가 어디에 장착되는지 작성
Grade: 해당 파츠의 등급

====스텟값====
----기초 스텟----
Weight: 무게, 이동속도에 영향줌, 전체 합산, 검에서는 검날이 손잡이보다 더 무거운 만큼 공격력 추가 / 후딜레이 추가
Damage: 기초공격력, 리시버의 기초값에 다른 파츠의 배율값을 곱해서 연산
RPM: 공격속도, 리시버의 기초값에 다른 파츠의 배율값을 곱해서 연산
Type: 공격방식, 서로 다른 공격방식끼리는 호환불가, NA는 상관없이 자유장착
----추가 피해----
C_Damage: 크리티컬 대미지, 전체 합산, 1이 기초공격력의 100%, 리시버에는 최소 1.0 권장
C_Chance: 크리티컬 확률, 전체 합산, 0아래로 내려갈시 0으로 조정됨, 1이 100%, 1위로 넘어갈시 항상 치명타
P_Percent: 관통 비율, 전체 합산, 공격력중 이 비율은 방어력을 무시하는 피해를 줌, 대신 이부분은 크리티컬 적용안됨
P_Amount: 관통 고정값, 전체 합산, 이 수치만큼의 대미지(관통비율 계산후 적용)만큼 관통피해로 전환됨, 크리티컬 적용안됨
----반동 제어----
Stability: 반동을 감소시키는 값, 리시버의 기초값에 다른 파츠의 배율값을 곱해서 연산
Accuracy: 기본 명중률 값, 리시버의 기초값에 다른 파츠의 배율값을 곱해서 연산
Recovery: 반동으로 인한 조준이탈상태에서 돌아오는 속도, 리시버의 기초값에 다른 파츠의 배율값을 곱해서 연산
----장탄 관련----
Rounds: 장탄수, 전체 합산
R_Speed: 재장전 속도, 리시버와 매거진의 기초값을 합친다음 다른 파츠의 배율값을 곱해서 연산
